# Vue Shopping Cart

![](screenshot.png)

## Technologies

- Deno.js
- Vue.js

## Database

- [] MongoDB

## Development

- [x] Deno.js server
- [x] Vue.js client

### Server

Go to server folder then running server with `https://localhost:3000`

```
$ cd server
$ npm start
```

### Client

Go to client folder then running client with `https://localhost:8080`

```
$ cd client
$ npm run dev
```
