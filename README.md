# Shopping Cart

Simple shopping cart with Vue.js and Node.js

![](screenshot.png)

## Usage

### Backend with Node.js

```shell
$ cd server
$ npm run dev
```

The first time you need seeders data for products and categories.

```shell
$ curl http://localhost:3000/api/seeders
```

### Client with Vue.js

```shell
$ cd client
$ npm run serve
```
